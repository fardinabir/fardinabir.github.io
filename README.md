# fardinabir.github.io

Personal portfolio of **Kazi Fardin Islam Abir** — Software Engineer (Go · Microservices · FinTech).

Live: https://fardinabir.github.io

## Stack
Vanilla HTML · CSS · JS. No build step. Hosted on GitHub Pages.

## Structure
```
index.html      # single-page site
styles.css      # styles
script.js       # interactions (typewriter, reveals, modal, counters)
resume.pdf      # downloadable résumé
assets/
  logos/        # company logos (experience timeline)
  people/       # recommender photos
.nojekyll       # disable Jekyll processing
```

## Local preview
```bash
python3 -m http.server 8000   # then open http://localhost:8000
```
